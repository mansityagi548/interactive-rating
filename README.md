# ⭐ Interactive Rating Component : 

An elegant and responsive interactive rating component built using **HTML, CSS, and JavaScript**.  

Users can select a rating, submit it, and instantly receive a thank you response with smooth UI interaction.

This project focuses on clean UI design, user interaction handling, and responsive frontend development practices.

## ✨ How It Works

1. User selects a rating from available options
   
2. Selected rating gets visually highlighted
   
3. User clicks the submit button
   
4. JavaScript captures the selected rating
   
5. The rating card dynamically changes into a thank-you state
   
6. Selected feedback is displayed to the user

## ✨ Features : 

✅ Interactive rating selection 

✅ Dynamic thank-you state  

✅ Responsive design  

✅ Clean and modern UI 

✅ Smooth user experience 

✅ Beginner-friendly code structure  

✅ DOM manipulation using JavaScript
